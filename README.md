## üöÄ Overview

This repository contains the source files and documentation for an interactive business intelligence dashboard designed to provide a deep, data-driven understanding of credit card activity.

The analysis is split into two core areas:
1.  **Transaction Analysis:** Focuses on revenue, transaction volume, interest, and spend patterns.
2.  **Customer Analysis:** Focuses on demographic and behavioral insights, including income, job function, education, and marital status.

This dashboard is a powerful tool for financial institutions to identify key revenue drivers, optimize marketing strategies, and manage risk.

## ‚ú® Key Features

The dashboard provides the following top-level insights:

* **Financial Metrics:** Tracking **Revenue** ($55.3M), **Transaction Amount** ($44.5M), **Total Interest** ($7.8M), and **Transaction Count** (655.7K).
* **Card Category Performance:** Detailed breakdown of performance across **Blue**, **Gold**, **Platinum**, and **Silver** card tiers.
* **Spending Patterns:** Analysis of revenue by **Chip Used** (Swipe, Chip, Online) and **Expenditure Type** (e.g., Bills, Entertainment, Fuel, Grocery).
* **Customer Demographics:** Insights into revenue and income by **Job Function** (e.g., Businessman, Blue-collar, Govt), **Education Level**, **Marital Status**, and **Age Group**.
* **Geographical Analysis:** Highlighting the **Top 5 States** contributing to revenue (TX, NY, CA, FL, NJ).
* **Time-Series Tracking:** Visualizations for **Revenue and Transaction Count by Quarter** and **Revenue by Week**.
